QUESTIONS:
1. What Nmap flag did you use to find the specific software versions running on these ports?
Answer:The -sV flag (Version Detection). This flag probes open ports to 
determine the exact service and version running on each one, 
such as "Apache httpd 2.4.41" or "OpenSSH 7.6p1". 

2. Why is it dangerous to run an intense scan (like -A) on a fragile legacy system?
Answer: Running an intense scan like -A on a fragile legacy system is 
dangerous because it sends aggressive and high-volume packet 
probes that can overwhelm older hardware or unstable services. 
Legacy systems often lack the resources to handle the flood of 
requests, which can cause crashes, service interruptions, or 
even permanent hardware damage. Additionally, -A enables OS 
detection, version scanning, script scanning, and traceroute 
simultaneously — a combination that can trigger intrusion 
detection systems or destabilize services that were never 
designed to handle modern network interrogation techniques.
