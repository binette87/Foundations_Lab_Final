## Security Foundations: Governance & Frameworks
The CIA triad are Confidentiality, Integrity and Availability. Confidentiality is protecting data from unauthorized access. Integrity is ensuring data is not altered. Availability is ensuring that data is accessible when needed.
The AAA Framework stand for Authentication, Authorization and Accounting. Authentication is to verify identity. Authorization is granting permissions. Accounting is keeping track of actions.
Governance, Risk, Compliance acts as the strategic layer and the blueprint that ensures security efforts meet business and regulatory goals. GRC is critical because it "bridges the gapbetween the server room and the boardroom" and "secures the budget for technical tools(National Institute of Standards and Technology.2024).

## Lab Infrastructure & Virtualization Setup
Definitions: 
Hypervisor - Software that creates and runs virtual machines (VMs) by isolating the computer's operating system and resources from the virtual environments.
Virtual Machine (VM) - A digital emulation of a physical computer that runs an operating system and applications just like a physical machine.
Isolation in cybersecurity prevents threats within one environment from spreading to others. This is critical for maintaining "virtual boundaries".
Virtualization and the CIA Triad:
virtual supports confidentiality by isolating data within specific VMs, Integrity by allowing for snapshots that revert systems to a known-good state if corrupted, and Availability by enabling the quick migration of resources to ensure systems remain accessible. (National Institute of Standards and Technology, 2024).
